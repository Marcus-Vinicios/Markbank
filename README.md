# Markbank
Projeto feito durante o curso de NodeJS da "Hora de Codar"

O projeto simula o sistema de um "caixa eletronico", permintindo o usuário consultar seu saldo, realizar um depósito ou um saque.
O que foi utilizado:
<ul>
  <li>JavaScript</li>
  <li>NodeJS</li>
  <li>Chalk</li>
  <li>Inquirer</li>
</ul>
